SUBIR ARCHIVOS A GITHUB DESDE BASH
1. Crear el repositorio en github
2. Click derecho sobre la carpeta raiz que contiene los archivos y carpetas del proyecto
3. Selecciona ejecutar bash here, luego en la consola:

git init # Inicializa el proceso
git status # verifica el estado del proyecto a subir
git add . # Agrega a red local los archivos contenidos en el directorio
git commit -m "mensaje" # Intenta subir el contenido, si no esta logeado, pedirá logeo
git branch -M main # subir el contenido a al remoto
git remote add origin https:ruta_del_proyecto
git push -u origin main # Activará sincronización del local con remoto

# Se puede actualizar la pagina de github, deberan aparecer los contenidos subidos

# Cuando se hagan cambios en el contenido del proyecto, para subirlos:

Ejecutar de nuevo el bash en la carpeta raiz

git status # para verificar cambios

PARA IGNORAR UN TIPO DE ARCHIVO:
1. En la raiz crear un archivo de texto con el nombre ".gitignore"
2. En el archivo escribir el tipo de archivo Ej. .class

git add . #agregar los archivos

git commit -m "otro mensaje"

git push -u origin main # Subirá los cambios hechos
